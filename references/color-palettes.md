# Color Palettes

Curated gradient pairs for app icon backgrounds, organized by mood. Each entry provides all colors needed for a complete icon.

## How to Use

1. Pick a palette that matches the app's mood/category
2. If the app has existing brand colors, derive a gradient from them instead
3. Each palette provides: gradient start (lighter), gradient end (darker), shadow color, foreground tint, face/detail stroke

## Deriving Custom Palettes

If the app has brand colors, derive a palette:

1. **Gradient Start**: The brand color at full saturation
2. **Gradient End**: Darken by 15-20% (reduce lightness in HSL)
3. **Shadow Flood**: Very dark version (10-15% lightness, same hue)
4. **Foreground Tint**: Very light version (90-95% lightness, same hue)
5. **Face/Detail Stroke**: Medium-dark version (30-40% lightness, same hue)
6. **Glow Accent**: Medium-light version (60-70% lightness, same hue)

Example: Brand color `#7C6BF0`
- HSL: 248, 82%, 68%
- Gradient End: 248, 75%, 60% = `#5A4BD6`
- Shadow: 248, 70%, 16% = `#1a0a4a`
- Tint: 248, 50%, 94% = `#EDE9FF`
- Stroke: 248, 28%, 49% = `#6B5CA0`
- Glow: 248, 60%, 84% = `#B8AEFF`
